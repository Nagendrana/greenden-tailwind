# greenden-tailwind
🌿 Greenden
A responsive plant e-commerce website built with HTML and Tailwind CSS, offering a curated selection of artificial and natural plants for home and office interiors.

🖥️ Live Pages

PageDescriptionindex.htmlHomepage — hero section, about us, best sellers, customer reviews, newsletterproduct.htmlProduct listing page with search bar and plant catalogcontact.htmlContact form with name, email, phone, subject, and message fields

✨ Features

Responsive Navbar — Hamburger menu on mobile, full nav on desktop (md: breakpoint)
Hero Section — Tagline, CTA buttons, and header image
About Section — Highlights large assortment, free/fast shipping, and 24/7 support
Best Sellers — Grid of 4 featured plants with hover animations
Product Catalog — 8-product grid with plant names and prices
Customer Reviews — Styled review cards with dark green background
Newsletter Signup — Email subscription input with subscribe button
Contact Form — Full-featured form with responsive 2-column layout and side image
Sticky Footer — Consistent branding across all pages


🛠️ Tech Stack

HTML5
Tailwind CSS v2.2.19 (via CDN)
